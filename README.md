# Groups Watching Ranges

### clone the repo, open a terminal window, cd into the project folder and run:
```shell
./main
```
### to easily send commands to the server, open a new terminal window in the same folder and run:
```shell
./client
```
### API:
```shell
ADD  start end name
DEL  start end name
DEL  start end
FIND value
FIND start end
```
